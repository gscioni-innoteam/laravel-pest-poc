### Utilizzo

#### Primo avvio
```shell
make build
```

#### Avviare i containers
```shell
make up
```

#### Fermare i containers
```shell
make stop
```

### Tools

#### Eseguire PhpStan
```shell
make stan
```

#### Eseguire il fix del coding style
```
make csfix
```

#### Eseguire la batteria di tests (funzionali ed unitari)
```shell
make test
```

#### Verificare il coverage
```shell
make coverage
```

#### Ottenere una lista dei comandi disponibili
```
make help
```
